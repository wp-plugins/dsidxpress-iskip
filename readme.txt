== Description ==

[dsIDXpress](http://wordpress.org/plugins/dsidxpress/) is the best WordPress IDX plugin. Wouldn't it be great if you could do more?

iSkip for dsIDXpress allows you to scroll through all of a listing's photos by simply hovering over the image on the search results page. __View the video below__ to see what a great feature this is.

[youtube http://www.youtube.com/watch?v=-wJ-zso8ofM]

It's a simple installation: simply have dsIDXpress running, and then install and activate this plugin.

Uses [iSkip](https://github.com/iantearle/iSkip) for the gallery functionality.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. No step 3!

== Frequently Asked Questions ==

= Do I need to be running dsIDXpress? =
Yes, you do.

= Do I need to be running IDX+? =
No, you don't need IDX+, but you should get it anyway!

= Will this work with dsIDXpress Lite or dsIDXpress Pro? =
Yes, this works for both dsIDXpress Lite and dsIDXpress Pro.
